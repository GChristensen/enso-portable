v1.4 July 18, 2026

  • Added voice recognition. Commands can be spoken instead of typed, prefixed by a keyword, e.g.: “computer open notepad”. Each command is enabled for voice individually on the Your Commands page, where it can also be hidden from the quasimode (voice-only) or set to ask for a spoken yes before it runs. Voice recognition can be suspended by a voice command or from the tray menu, and stops by itself while the workstation is locked. See the tutorial for details.
  • Experimental Linux port, supporting X11 sessions and Wayland on KDE Plasma. See README.linux.md.
  • Experimental macOS port, with the same feature set as the Linux port. See README.macos.md.

v1.3 October 7, 2025

  • Migrated to Python 3.14.
